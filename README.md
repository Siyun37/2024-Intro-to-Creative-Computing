## Project 04 How to drink coffee in space

<img height="300" alt="image" src="https://github.com/user-attachments/assets/87b40447-1c45-4514-a782-12605e13f994" />


1. Coffee drops start to move inside the cup, and they are moving randomly.
2. The color of the drop is based on the coffee color, with random opacity.
3. When drop moves inside the cup, it stops moving.
4. When drop moves outside the cup, it starts moving again.
5. When mouse pressed the drop, user can grab the drop and move it to another place.
6. When mouse clicked on the drop, it splashes into 10 smaller drops.

## Project 10 Public bicycle rental status

<img height="400" alt="image" src="https://github.com/user-attachments/assets/575758de-ae5c-4cd2-9044-d940df3926ab" />


1. Circles with different sizes to visualize the number of bicycles.
2. Each circle has a transparency of 50%, so it can be distinguished from each other.
3. When you move the mouse in a circle, the number of bicycles is displayed in the center of the circle.
4. When you click the mouse, the circles are removed and the bicycles are displayed.
5. Different sizes of bicycles are visualizing the number of bicycles.
6. When you move the mouse in a circle, small bicycles are displayed.
7. The number of small bicycles is the number of bicycles in the place.
8. If you click the mouse again, you can go back to the first scene.

https://editor.p5js.org/tldbs37/sketches/lLC6hTDlS
