## Project 04 How to drink coffee in space

<img height="300" alt="image" src="https://github.com/user-attachments/assets/87b40447-1c45-4514-a782-12605e13f994" />


1. Coffee drops start to move inside the cup, and they are moving randomly.
2. The color of the drop is based on the coffee color, with random opacity.
3. When drop moves inside the cup, it stops moving.
4. When drop moves outside the cup, it starts moving again.
5. When mouse pressed the drop, user can grab the drop and move it to another place.
6. When mouse clicked on the drop, it splashes into 10 smaller drops.

https://editor.p5js.org/tldbs37/full/eIAfbgWti
